Getting started

First things first you'll need to clone the source code of this SPA(Single page application) that is found here
Once you have this you should navigate to the following directory:
cd ~/repo/automation-sandbox
once there you should run
npm install
Once that complete's you should be ready to run the UI Automation, to do this run the following
ng e2e
If that's succeeded you should have a list of test results output

Writing a test

The tests are stored in the "e2e" folder next to the "src" folder
